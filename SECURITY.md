# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x.x   | Yes       |

## Reporting a Vulnerability

To report a security vulnerability in SudoWP Log Viewer, use GitHub's private
vulnerability reporting on this repository, or contact security@sudowp.com.

Do not open a public GitHub issue for security vulnerabilities.

We acknowledge receipt within 48 hours and aim to release a patch within 14 days
for confirmed critical issues.
